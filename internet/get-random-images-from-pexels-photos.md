# Obtenha imagens aleatórias do rseat.pics

_O exemplo abaixo não funciona no GitHub. Parece que o GitHub faz cache e serve sua própria versão da imagem. Isso funcionará em qualquer lugar onde você não esteja fazendo cache de imagens._

Aqui está uma imagem do [rseat.pics](https://rseat.pics).

![imagens aleatórias](https://rseat.pics)

Eu não sei qual imagem estou mostrando para você. Isso porque a URL usada para a imagem acima é `https://rseat.pics`. Isso instrui o Unsplash a nos fornecer aleatoriamente uma imagem de `1200x600`. Tente atualizar a página e você verá que a imagem é diferente a cada vez. Legal, né?
