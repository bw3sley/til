# Como Pesquisar no Google

Este é um artigo que aborda algumas maneiras de melhorar seu método de busca no Google.

## Removendo palavras da pesquisa

Coloque um `-` antes de uma palavra para removê-la dos resultados.

Exemplo:  

`como obter o valor de um input -jquery`

## Pesquisando por uma expressão exata

Coloque uma palavra ou frase entre aspas para encontrar resultados que correspondam exatamente à expressão.

Exemplo:  

`"cannot read property 'map' of undefined"`

## Combinando pesquisas

Coloque `OR` ou `|` entre as expressões que deseja combinar.

Exemplo:  

`maratona OR corrida` ou `(maratona | corrida) sprint`

## Pesquisando em um site específico

Coloque `site:` antes de uma URL para encontrar resultados apenas de um site específico.

Exemplo:  

`site:fireship.io react hooks`

## Pesquisando por resultados relacionados

Coloque `related:` antes de uma URL para encontrar resultados relacionados ao site.

Exemplo:  

`related:time.com`

## Visualizando a versão em cache do Google de um site

Coloque `cache:` antes de uma URL para ver a versão em cache do Google daquele site.

Exemplo:  

`cache:www.rocketseat.com.br`

## Pesquisando após ou antes de uma data específica

Use `after:` ou `before:` para encontrar resultados mais novos ou mais antigos que uma determinada data.

Exemplos:  

`react hooks after:2021`  
`react hooks before:2021`

## Pesquisando por um período de tempo

Coloque `..` entre dois números ou datas para encontrar resultados dentro desse intervalo.

Exemplo: 
 
`anúncio angular 2 2013..2015`

## Pesquisando por um tipo de arquivo

Use `filetype:` após uma palavra ou frase para encontrar arquivos específicos.

Exemplo:  

`código limpo filetype:pdf`

## Pesquisando usando curingas

Use o operador `*` para substituir qualquer palavra ou frase e encontrar resultados variados.

Exemplos:  

`site:*fireship.io -www`  
`como criar * com tailwindcss`
