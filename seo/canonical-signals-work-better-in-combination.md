# Canonicalização fica mais forte com sinais combinados

Quando existe risco de conteúdo duplicado, vale combinar sinais em vez de depender de uma única pista para canonicalização.

`rel="canonical"`, sitemap e redirects apontando para a mesma URL deixam essa indicação mais consistente.

```html
<link rel="canonical" href="https://example.com/dresses/green-dresses" />
```
