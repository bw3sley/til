# Usar `<del>` para marcar texto removido

`<del>` indica que um trecho foi removido ou deixou de ser válido.

```html
<p>De <del>R$ 100,00</del> por R$ 80,00.</p>
```

// O valor "R$ 100,00" aparece riscado.
