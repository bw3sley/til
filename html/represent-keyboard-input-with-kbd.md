# Usar `<kbd>` para representar entrada de teclado

`<kbd>` marca teclas, atalhos e comandos digitados pelo usuário em documentação ou interfaces.

```html
<p>Pressione <kbd>Ctrl</kbd> + <kbd>S</kbd>.</p>
```

// Renderiza atalho de teclado: Ctrl + S
