# Usar `<details>` e `<summary>` para conteúdo expansível

`<details>` cria uma área expansível nativa, e `<summary>` define o rótulo clicável que abre e fecha esse conteúdo.

```html
<details>
  <summary>Ver mais</summary>
  <p>Conteúdo adicional.</p>
</details>
```
