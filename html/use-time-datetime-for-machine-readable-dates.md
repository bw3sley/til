# Usar `<time>` com `datetime` para datas legíveis por máquina

`<time>` pode exibir uma data formatada para pessoas e, ao mesmo tempo, manter um valor estruturado em `datetime`.

```html
<time datetime="2025-05-20">20 de maio de 2025</time>
```

// Exibe: 20 de maio de 2025
