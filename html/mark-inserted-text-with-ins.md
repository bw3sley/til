# Usar `<ins>` para marcar texto inserido

`<ins>` indica que um trecho foi adicionado ao conteúdo. Isso faz mais sentido quando a inserção precisa ter valor semântico, não só visual.

```html
<p>Agora está por <ins>R$ 80,00</ins>.</p>
```

// O valor "R$ 80,00" aparece como conteúdo inserido.
