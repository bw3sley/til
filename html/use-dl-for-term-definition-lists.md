# Usar `<dl>` para listas de termos e definições

`<dl>` representa uma lista de descrições. Use `<dt>` para o termo e `<dd>` para a definição.

```html
<dl>
  <dt>HTML</dt>
  <dd>Linguagem de marcação.</dd>
</dl>
```
