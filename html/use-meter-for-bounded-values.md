# Usar `<meter>` para valores dentro de um intervalo

`<meter>` representa um valor medido dentro de um intervalo conhecido, como uso de disco ou nota.

```html
<meter value="70" min="0" max="100">70%</meter>
```

// Exibe medidor preenchido em 70%.
