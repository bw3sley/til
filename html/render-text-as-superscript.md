# Renderizar Texto como Sobrescrito

Existe uma [tag HTML](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/sup) que instrui os mecanismos dos navegadores a renderizar texto como sobrescrito — como quando você precisa de um expoente.

```html
<p>Renderize este texto como <sup>sobrescrito</sup>!</p>
```

Aqui está como o exemplo acima ficaria:

<p>Renderize este texto como <sup>sobrescrito</sup>!</p>

Confira este [exemplo no CodePen](https://codepen.io/jbranchaud/pen/GzoyXe) para experimentar.
