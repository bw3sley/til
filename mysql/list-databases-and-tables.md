# Usar `SHOW DATABASES` e `SHOW TABLES` no MySQL

Quando você está no cliente MySQL, `SHOW DATABASES` lista bancos disponíveis e `SHOW TABLES` lista tabelas do banco atual.

```sql
SHOW DATABASES;
USE my_app_dev;
SHOW TABLES;
```
