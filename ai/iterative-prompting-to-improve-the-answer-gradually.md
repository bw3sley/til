# Iterative Prompting para melhorar a resposta aos poucos

Iterative Prompting ajusta a saída em ciclos curtos, mantendo o contexto da conversa. É útil quando a resposta ficou próxima do ideal, mas ainda precisa de refinamento.

Exemplo:

```txt
Reescreva esse texto de forma mais objetiva.
Adicione exemplos práticos no contexto de marketing.
Reduza para até 80 palavras, mantendo o ponto principal.
Agora adapte esse conteúdo para gestores.
```
