# Refinamento de Perguntas para reduzir respostas genéricas

Refinamento de Perguntas melhora uma pergunta inicial antes de pedir a resposta final. Isso ajuda quando o pedido está vago, amplo demais ou mal direcionado.

Exemplo:

```txt
Refine essa pergunta para torná-la mais clara: Como usar o ChatGPT no trabalho?
```
