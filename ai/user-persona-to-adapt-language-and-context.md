# Persona do Usuário para adaptar linguagem e contexto

Persona do Usuário inclui no prompt o perfil de quem vai consumir a resposta. Isso faz a IA ajustar vocabulário, profundidade e exemplos para aquele público.

Exemplo:

```txt
Explique o que é ROI para um empreendedor iniciante, usando linguagem simples e exemplos práticos do dia a dia.
```
