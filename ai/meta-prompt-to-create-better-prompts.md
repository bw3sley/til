# Meta-Prompt para criar prompts melhores

Meta-Prompt é um prompt usado para escrever, revisar ou melhorar outro prompt. É útil quando você sabe o resultado que quer, mas ainda não encontrou a melhor forma de pedir isso para a IA.

Exemplo:

```txt
Me ajude a criar um prompt para gerar ideias de post no LinkedIn voltado para empreendedores iniciantes na área de tecnologia.
Quero que as respostas sejam diretas, com dicas práticas e exemplos reais.
```
