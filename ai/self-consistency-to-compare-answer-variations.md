# Self-Consistency para comparar variações de resposta

Self-Consistency pede várias respostas para o mesmo problema e depois compara as opções. Isso ajuda quando existem caminhos diferentes e você quer avaliar qual faz mais sentido.

Exemplo:

```txt
Como melhorar o engajamento de clientes em um e-commerce?

Responda com 3 abordagens diferentes (marketing, UX e pós-venda).

Depois, diga qual estratégia é mais indicada para empresas com orçamento limitado e justifique.
```
