# Prompt de Follow-Up para refinar resposta anterior

Prompt de Follow-Up continua uma conversa anterior para ajustar, aprofundar ou corrigir uma resposta sem recomeçar do zero. Ele aproveita o contexto que a IA já recebeu.

Exemplo:

```txt
Crie um post para LinkedIn divulgando que estou me especializando em Inteligência Artificial.

Agora adapte esse texto para um tom mais profissional e adicione um exemplo concreto de aplicação no trabalho.
```
