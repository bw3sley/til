# Ancoragem de Memória para manter o mesmo padrão

Ancoragem de Memória relembra contexto, estilo ou estrutura já definidos antes. Ela serve para manter consistência ao longo de uma conversa com várias etapas.

Exemplo:

```txt
Lembrando que estamos escrevendo para crianças de 10 anos, com linguagem simples.
Use o mesmo estilo definido anteriormente.
Continue a partir do que foi criado, sem mudar o padrão.
```
