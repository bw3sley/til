# Prompt Layering para quebrar tarefas complexas

Prompt Layering divide uma tarefa complexa em camadas sequenciais. Cada etapa prepara a próxima, o que dá mais controle sobre a construção da resposta.

Exemplo:

```txt
Etapa 1: Liste como usar Inteligência Artificial voltado para profissionais de marketing digital.

Etapa 2: Detalhe os tópicos da etapa anterior com foco em aplicações práticas no planejamento, execução e análise de campanhas.

Etapa 3: Organize tudo no formato para enviar por e-mail.
```
