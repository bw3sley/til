# Zero-Shot Prompting para pedidos diretos

Zero-Shot Prompting é quando você pede algo para a IA sem fornecer exemplos antes. Funciona melhor em tarefas simples, claras e com pouca necessidade de controlar formato ou estilo.

Exemplo:

```txt
Explique o que é funil de vendas para que eu possa apresentar isso rapidamente a um cliente leigo.
```
