# Usar `crypto.randomUUID()` para gerar UUID v4 no navegador

`crypto.randomUUID()` gera UUID v4 aleatório sem biblioteca externa em navegadores modernos.

```js
crypto.randomUUID();
// "f2050c5e-af52-4ca2-b4d6-23758b3396c9"
```
