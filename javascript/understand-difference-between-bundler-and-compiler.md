# Diferenciar bundler de compiler no ecossistema frontend

Bundler junta vários arquivos em um ou mais bundles. Compiler transforma sintaxe ou código-fonte em uma versão compatível com outro alvo.

Na prática, bundler organiza entrega do projeto. Compiler adapta linguagem ou recurso para rodar no ambiente desejado.
