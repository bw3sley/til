# SP_HELP

No **SQL Server**, você pode usar procedimentos armazenados do sistema para obter informações detalhadas sobre tabelas e colunas.

## SP_HELP

O procedimento armazenado **sp_help** fornece informações gerais sobre qualquer objeto no banco de dados, como tabelas, views ou procedures.

Exemplo:

```sql
SP_HELP Customer;
```

Esse comando exibe informações como estrutura da tabela, restrições, índices e muito mais.
