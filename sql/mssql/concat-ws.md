# CONCAT_WS

A função `CONCAT_WS()` concatena duas ou mais strings utilizando um separador definido entre elas.

Exemplo:

```sql
SELECT CONCAT_WS('.', 'www', 'W3Schools', 'com'); -- www.w3schools.com
```

Neste exemplo, o ponto (`.`) é usado como separador entre as strings.
