# Usar DML para inserir, atualizar e remover dados

DML reúne comandos que alteram dados existentes, como `INSERT`, `UPDATE` e `DELETE`.

```sql
UPDATE Cliente
SET Email = 'exemplo@exemplo.com'
WHERE Id_Cliente = 1;
```
