# Usar DDL para criar, alterar e remover estruturas

DDL cobre comandos de estrutura como `CREATE`, `ALTER`, `DROP` e `TRUNCATE`.

```sql
CREATE TABLE Clientes (
  Id BIGINT PRIMARY KEY,
  NomeCliente VARCHAR(50)
);
```
